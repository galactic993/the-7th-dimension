# 星のステージ - 要件定義書

「星のステージ」は、中村咲太が主催する7次元ユニコーンワークショップの参加者が得た高次元の気づきを安全かつ自由に共有し合えるInstagramキュレーションサイトです。

## プロジェクトビジョン
- 参加者が7次元ユニコーンワークショップで得た高次元の気づきを安全かつ自由に共有し合い、コミュニティ全体の意識進化を加速させる。
- エネルギー体験を視覚・言語の両面でアーカイブし、後から振り返ることで学びを深化させる。

## 目的（ゴール）
1. ワークショップ参加者が自身の体験を投稿・保存できる場を提供する。
2. 同様の体験を持つユーザー同士がインスピレーションを与え合えるコミュニティを形成する。
3. ワークショップのブランド価値をオンライン上で継続的に拡散・向上させる。

## ターゲットユーザ
| ペルソナ | 主なニーズ | 7次元体験レベル |
|---------|-----------|----------------|
| 初心者: 『参加したばかり』 | 体験を言語化・共有する方法を知りたい | ★☆☆ |

## MVP範囲
以下の機能は初回リリース(MVP)で必須とする。
1. 投稿グリッド表示
2. 投稿詳細モーダル
3. 全文検索＋ランダム表示
4. いいね・保存機能
5. モックデータでの動作確認
6. レスポンシブデザイン

> **除外**: コメント投稿、新規投稿作成、外部API連携などはポストMVPとする。

## 主要機能

### 1. インスピレーション・ギャラリー機能
- **投稿グリッド**: Instagram風のカード形式で、参加者から寄せられた様々なインスピレーション（アート、文章、動画など）を表示します。
- **レスポンシブデザイン**: デバイスに応じて2-5列の可変グリッドで最適表示。
- **ホバーエフェクト**: マウスオーバー時に、投稿へのリアクション数（いいね）、コメント数、投稿者情報、関連タグをオーバーレイで表示。

### 2. 検索・フィルタリング機能
- **全文検索**: キャプション、ユーザー名、表示名、タグ、位置情報など、投稿に含まれるテキスト情報から横断的に検索。
- **ランダム表示（シャッフル）**: 投稿をランダムに並び替え、新たなインスピレーションとの偶然の出会いを創出する。セレンディピティを重視した主要機能。
- **リアルタイム検索**: 入力と同時に検索結果を動的にフィルタリングし、ストレスのない探索体験を提供。

### 3. 投稿詳細機能
- **モーダル表示**: グリッド上の投稿をクリックすると、詳細情報をモーダルウィンドウで表示。
- **リッチコンテンツ表示**: 高解像度画像、動画、音声コンテンツの再生、フォーマットされたテキストの表示。
- **投稿情報**: キャプション、タイムスタンプ、位置情報、投稿者情報などを分かりやすく表示。

### 4. ユーザーインタラクション機能
- **いいね機能**: ハートアイコンで直感的に共感や応援のエネルギーを送る機能。
- **保存機能**: 気になった投稿をブックマークし、後から簡単に見返せるようにする機能。
- **コメント表示**: 投稿に寄せられたコメントを表示（コメント投稿機能は将来拡張）。

### 5. 投稿・認証機能
- **Instagram連携**:
    - 公式ハッシュタグ `#the7thdimension` を付けてInstagramに投稿されたコンテンツを、自動的にサイトに連携・表示。
    - 投稿の真正性を担保するため、事前に参加者のInstagramアカウントをホワイトリスト形式で登録。
- **直接投稿**:
    - Instagramを利用していない参加者のため、メールアドレス認証を経てサイトから直接コンテンツ（画像、テキスト）を投稿できるフォームを提供。

### 6. ユーザー情報
- **プロフィール情報**: アバター、ユーザー名、表示名を表示。
- **認証バッジ**: 中村咲太氏や関係者など、公式なアカウントに青いチェックマークを表示。

## UI/UXデザイン

### カラーパレット
- **プライマリ**: 神聖さとエネルギーを表現するグラデーション（黄色→ピンク→紫）。
- **背景**: 宇宙空間や高次元をイメージさせる、淡いグレーやダーク系のグラデーション。
- **アクセント**: ピンク、ゴールド（フォーカス、インタラクション時）。

## コンテンツ特性

### テーマ
- 7次元ユニコーンワークショップでの体験
- 7次元の自己、多次元意識との繋がり
- ユニコーンエネルギーとの共同創造
- 沈黙の中で受け取ったインスピレーション
- エネルギーレベルでの変容と目醒めの記録
- 意識の統合プロセス

### ハッシュタグ
- **公式**: `#the7thdimension`
- **関連**: `#7次元ユニコーン`, `#多次元意識`, `#ユニコーンヒーリング`, `#沈黙のワークショップ`, `#エネルギー変容`, `#意識の目覚め`, `#インスピレーションアート`, `#次元統合`

### 投稿内容の多様性
- ヒーリング中に描いた絵、スケッチ、デジタルアート
- 受け取ったメッセージを綴った詩や文章
- インスピレーションから生まれた楽曲や演奏（音声・動画）
- エネルギーを表現した舞や踊り、パフォーマンス（動画）
- 体験から得た深い気づきや変容のレポート

## 技術要件

### フロントエンド技術スタック
- **React 18** + TypeScript
- **Vite** (ビルドツール)
- **Tailwind CSS** (スタイリング)
- **Lucide React** (アイコン)

### 状態管理
- React Hooks (useState, useMemo) を中心としたシンプルなローカル状態管理。

### 外部サービス連携
- **Instagram Graph API**: 特定ハッシュタグの付いた投稿を定期的に取得するために利用。

### 認証
- **Instagramアカウント認証**: 投稿連携を行うユーザーの事前認証。
- **メールアドレス認証**: 直接投稿機能のための本人確認。
